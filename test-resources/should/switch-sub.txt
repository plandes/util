Usage: python -m unittest <emp|org> [options]:

An employee database application (print using "python -m unittest emp -d").

Options:
  -h, --help [actions]    show this help message and exit
  --version               show the program version and exit
  -c, --config FILE       the configuration file

Actions:
emp <format>              output employee by id with <format> (should not replace "org_id")
  format <json|txt>       output format for employee with ID given by -e
  -d, --dryrun            whether to execute an action
  -e, --empid INT         the id of the employee

org                       process organizations by id or print with -d (should replace -o)
  -d, --dryrun            whether to execute an action
  -o, --orgid INT         a unique id
exit: 0
