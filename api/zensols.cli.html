zensols.cli.peraction module¶

More advanced implementation of an action based command line.

class zensols.cli.peraction.OneConfPerActionOptionsCli(opt_config, config_type=<class 'zensols.config.iniconfig.IniConfig'>, **kwargs)[source]¶

Bases: zensols.cli.peraction.PerActionOptionsCli

Convenience action handler that allows a definition on a per action basis. See the test cases for examples of how to use this as the detail is all in the configuration pased to the init method.

Parameters

  • opt_config – the option configuration (see project documentation)

  • config_type – the class used for the configuration and defaults to zensols.util.configbase.Configurable.

  • config – an instance of zensols.config.Config

  • version (str) – the default version of this command line module, which is overrided by the package’s version if it exists

  • opts (set) – options to be parsed

  • manditory_opts (set) – options that must be supplied in the command

  • environ_opts (set) – options to add from environment variables; each are upcased to be match and retrieved from the environment but are lowercased in the results param set

  • default_action (str) – the action to use if non is specified (if any)

_config_log_level(fmt, levelno)[source]¶

Configure logging by calling logging.basicConfig() with the base package set at levelno if available. In this case, the default logging level is set to logging.WARNING. Otherwise, reconfigure logging using a levelno across all loggers.
